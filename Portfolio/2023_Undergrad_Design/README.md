# 神经辐射场在青瓷艺术创作领域中的设计应用 (2023 本科毕业设计)

## 项目概要
本项目是陈禹汀的本科毕业设计（工业设计方向），致力于将前沿的 **神经辐射场技术（NeRF）** 引入传统的 **青瓷艺术创作** 领域。针对现有陶瓷辅助软件无法真实还原釉面复杂光学特性（如折射、多层肌理）以及忽视烧制过程“随机性”的痛点，提出了一套基于 NeRF 的数字化解决方案，实现了从实体釉片到数字资产的高保真重建与可编辑预测。

## 项目主要内容

### 1. 核心技术：NeRF-Tex
- **原理**：利用神经网络渲染技术，从照片像素中学习釉面的体素信息，不仅还原几何形状，更精确捕捉光线在釉层中的传播效果。
- **数据流**：建立“配釉成分 -> 实际烧制 -> 扫描数据 -> 模型训练 -> 渲染预测”的闭环。

### 2. 硬件创新：自动化扫描台
- **设计**：自主研发了一套专用于陶瓷釉片的小型自动化扫描设备。
- **结构**：包含顶部扫描基盘、瓷片限位机构、云台电机和电子控制仓。
- **功能**：通过控制转台匀速转动，自动采集多角度图像数据，降低了 NeRF 高质量数据集的采集门槛。

### 3. 软件系统：青瓷配釉辅助平台
- **功能模块**：
    - **配釉主界面**：数字化调整釉料配方。
    - **釉面数据库**：存储与检索不同配方的烧制效果。
    - **渲染预览**：基于 NeRF 模型实时预测特定光照和视角下的釉面质感。
- **合作背景**：与龙泉市首席技师吴育敏老师合作，建立基于传统龙泉青瓷釉方的高质量数据集。

## 核心的设计思考
- **连接“数字”与“物质”**：AIGC（如 Midjourney）通常只能生成二维图像，无法处理陶瓷釉面这种具有深度的复杂材质。本项目试图填补这一空白，让 AI 理解“材质”而非仅仅是“图像”。
- **接纳“随机性”**：传统 CAD 追求完全确定的结果，但陶瓷艺术的魅力在于“窑变”的随机性。通过学习大量样本，模型试图捕捉并模拟这种随机的美感，而非简单的纹理贴图。
- **服务设计视角**：明确目标用户为初级艺术创作者、手工艺从业者和研究者，旨在通过数字化工具降低试错成本，提升沟通效率。

## 项目内很出彩的内容
- **跨学科融合**：将计算机视觉领域的 NeRF 技术硬核地应用到传统非遗技艺（青瓷）中，展现了极强的技术落地能力。
- **自研硬件设备**：为了解决数据集获取难的问题，动手制作了自动化扫描转台，体现了工业设计的工程素养。
- **高保真 UI 设计**：软件界面不仅功能完善，且视觉呈现专业，特别是“釉面条件筛选”和“配釉主界面”的设计，直观易用。

# 2023 Design Portfolio - Chen YuTing

## 项目概要
本作品集汇集了 Chen YuTing 在 2023 年期间的交互设计与工业设计作品，涵盖了从硬件系统、移动应用到思辨设计（Speculative Design）和互动装置等多个领域。作品核心关注点在于通过技术手段解决实际问题（如近视、直播生产力）以及对未来社会现象（如内卷、信息扭曲）的深刻反思与批判性设计。

## 项目主要内容

### 1. Dimension Smasher (维度粉碎者)
- **内容**：一套软硬件结合的系统，旨在扩展直播主的创造力空间，降低虚拟制片（Virtual Production）的门槛。
- **核心功能**：
    - **自动化光照匹配**：解决绿幕反光和光照不匹配问题。
    - **实时合成**：无需复杂的后期流程，支持手持运镜的无限移动。
    - **硬件模块**：包含激光雷达（LiDAR）、传感器、控制拨轮等，支持深度检测和自动对焦。
- **设计亮点**：打破大公司对虚拟制片资源的垄断，赋予个人创作者“好莱坞级”的生产力。

### 2. Myo-Clear APP
- **内容**：利用移动设备（手机/平板）的 LiDAR 和计算机视觉技术，预防青少年近视的应用程序。
- **核心功能**：
    - **高精度测距**：利用 LiDAR 实现毫米级（±1mm）的眼屏距离检测。
    - **直观反馈**：当距离过近时，屏幕会自动变模糊（Blur Reminder），而非单纯的弹窗警告，通过视觉直觉引导用户调整姿势。
    - **数据监测**：建立家长与孩子间的视力健康数据桥梁。

### 3. Enhanced Spine (强化脊柱 - 思辨设计)
- **内容**：针对未来“内卷”社会设计的脊柱增强装置。设定在过度追求效率的社会，年轻人的脊柱健康成为巨大问题。
- **核心功能**：
    - **三阶段治疗/增强**：从单纯的支撑（Pain Onset），到肌肉干预（Muscle），最终到植入式增强（Implant）。
    - **能量回收**：利用脊柱运动切割磁感线进行能量回收。
    - **力反馈支撑**：提供物理支撑并纠正体态。
- **设计反思**：“现在的年轻人不需要按摩，他们需要正骨”。通过夸张的义体化设计，反讽当代社会对效率的过度追求和对身体的异化。

### 4. Detective Rhetoric (侦探修辞)
- **内容**：一个交互式机器人装置，旨在唤醒人们对聊天机器人（Chatbot）信息扭曲的警惕。
- **核心功能**：
    - **信息扭曲模拟**：基于香农-韦弗通讯模型，可视化信息在传播过程中的“噪音”和“扭曲”。
    - **人机对抗**：观众可以通过物理装置（拉杆、选择器）尝试修正信息，但机器会根据算法（LSTM模型）的“注意力”强制填充和扭曲历史。

### 5. 其他探索项目
- **Untitled Quantum Game**：使用 Unity 引擎开发的量子科学游戏，可视化波函数、位置与动量的不确定性。
- **Lightime**：基于 FOC（磁场定向控制）和深度检测的桌面分区照明台灯。
- **Material Movement**：再生材料记录设备与艺术互动装置。

## 核心的设计思考
- **技术平权**：在 *Dimension Smasher* 中，致力于将昂贵的工业级技术（虚拟制片）下放给个人创作者。
- **直觉化交互**：在 *Myo-Clear* 中，放弃说教式的警告，转而使用生理本能（看不清就离远点）的视觉反馈。
- **批判性反思**：*Enhanced Spine* 和 *Detective Rhetoric* 不仅是解决问题的工具，更是提出问题的媒介，探讨技术异化、社会压力和信息真实性。

## 项目内很出彩的内容
- **Enhanced Spine 的世界观设定**：构建了一个完整的未来医疗消费陷阱——从简单的止痛贴片一步步诱导用户进行不可逆的机械植入，深刻揭示了消费主义与健康焦虑的结合。
- **Myo-Clear 的“模糊提醒”机制**：巧妙利用屏幕模糊这一负面反馈作为矫正手段，既符合近视的隐喻，又具有极强的行为引导效果。
- **Dimension Smasher 的硬件整合**：将 LiDAR、UWB 等先进传感器集成到模块化相机附件中，展现了极强的工程落地能力和硬件设计美学。
